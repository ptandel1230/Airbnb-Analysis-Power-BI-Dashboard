# Airbnb-Analysis-Power-BI-Dashboard
Airbnb data analysis using Power BI 
## Project Overview
This project analyzes Airbnb listings in Berlin to identify pricing trends,
booking activity, and property distribution across neighborhoods.

## Tools Used
- Power BI (dashboarding & DAX)
- Excel / CSV (data source)
- SQL (conceptual data cleaning & aggregation layer)

## Workflow
1. Raw Airbnb data sourced from Excel/CSV
2. (Optional) SQL used for data cleaning, filtering, and aggregation
3. Cleaned data visualized in Power BI using interactive dashboards

## Key Insights
- Apartments dominate total listings
- Average price varies significantly by property type
- Booking activity peaks mid-year
- Certain neighborhoods generate higher revenue

## Dashboard Features
- KPI cards (Total Revenue, Total Bookings)
- Property type distribution
- Average price per night
- Neighborhood map
- Interactive slicers (Year, Neighborhood, Room Type)

## Files
- `powerbi/` – Power BI dashboard file
- `sql/` – SQL scripts for data preparation
- `data/` – Dataset (or sample)
- `screenshots/` – Dashboard images
